1. Дискретное распределение Пуассона¶

: $ p(x)=\dfrac{\lambda^x}{x!}e^{-\lambda},\; \lambda>0 $

1.1 Описание основных математических характеристик¶
: $E\xi=\lambda$
: $ D\xi=\lambda $
: $ \varphi_\xi(t)=e^{\lambda(e^{it}-1)} $
: $ \varphi_\lambda(t)=e^{\lambda(t-1)} $

Выводы:¶

  • $E\xi=\sum\limits_{k=0}^{\infty}k\dfrac{\lambda^ke^{-\lambda}}{k!}=\sum\limits_{k=0}^{\infty}\dfrac{\lambda^ke^{-\lambda}}{(k-1)!}= \sum\limits_{k=0}^\infty\dfrac{k\lambda^k\sum\limits_{n=0}^{\infty}\frac{-\lambda^n}{n!}}{k!}= e^{-\lambda}\sum\limits_{k=0}^\infty\dfrac{\lambda^k}{(k-1)!}=\{j = k-1\} e^{-\lambda}\lambda\sum\limits_{j=0}^{\infty}\dfrac{\lambda^j}{j!}= e^{-\lambda}\lambda e^\lambda=\lambda$

  • $ E(\xi)^2=\sum\limits_{k=0}^{\infty}k^2\dfrac{\lambda^ke^{-\lambda}}{k!}= \sum\limits_{k=0}^{\infty}k\dfrac{\lambda^ke^{-\lambda}}{(k-1)!}= \sum\limits_{k=0}^\infty\dfrac{(k-1)\lambda^k}{(k-1)!}e^{-\lambda} + \sum\limits_{k=1}^\infty\dfrac{\lambda^k}{(k-1)!}e^{-\lambda}= \sum\limits_{k=2}^\infty\lambda^2e^{-\lambda}\dfrac{\lambda^{k-2}}{(k-2)!}+E\xi= \lambda^2e^{-\lambda}e^\lambda+\lambda=\\ =\lambda^2+\lambda $

  • $ D\xi=E(\xi)^2-(E\xi)^2=\lambda^2+\lambda-\lambda^2=\lambda $

  • $ \varphi_\xi(t)=\sum\limits_{k=0}^\infty e^{itk}\dfrac{\lambda^k}{k!}e^{-\lambda}=e^{-\lambda}\sum\limits_{k=0}^\infty\dfrac{(e^{it}\lambda)^k}{k!}=e^{-\lambda}e^{e^{it}\lambda}=e^{\lambda(e^{it}-1)} $

  • $ \varphi_\lambda(t)=\sum\limits_{k=0}^\infty\dfrac{\lambda^k}{k!}e^{-\lambda}t^k=e^{-\lambda}\sum\limits_{k=0}^\infty\dfrac{(\lambda t)^k}{k!}=e^{-\lambda}e^{\lambda t}=e^{\lambda(t-1)} $

import random
import math

def rand_Pois(lamb, n):
    nums = []
    for i in range(n):
        p = math.exp(-lamb)
        sp = 0
        sp += p
        u = random.uniform(0,1)
        j = 0
        while u > sp:
            j += 1         # p_j
            p *= lamb / j  # p_j / p_j-1 = lamb / j
            sp += p
        nums.append(j)
    return nums

2.1 Описание основных математических характеристик¶
: $ 1-exp\left(-\dfrac{x^2}{2\sigma^2}\right) $
$ \sqrt{\dfrac{\pi}{2}}\sigma $
: $ \left(2-\dfrac{\pi}{2}\right)\sigma^2 $
: $ 1 + t\sigma\sqrt{\dfrac{\pi}{2}}e^{\frac{\sigma^2 t^2}{2}}\cdot\left[\dfrac{2}{\sqrt{\pi}}\int\limits_0^{\frac{t \sigma}{\sqrt{2}}} e^{-t^2}dt +1 \right] , \;\; \dfrac{2}{\sqrt{\pi}}\int\limits_0^{\frac{t \sigma}{\sqrt{2}}} e^{-t^2}dt = erf\left(\dfrac{t \sigma}{\sqrt{2}}\right)$

Выводы:¶

  • $E\xi = \int\limits_0^\infty x \dfrac{x}{\sigma ^ 2} e^{-\frac{x^2}{2\sigma^2}} dx = \dfrac{1}{\sigma^2}\int\limits_0^\infty x \cdot x \cdot e^{-\frac{x^2}{2\sigma^2}} dx = \left\{u = x, \;du = dx, \; v = \int x e^{-\frac{x^2}{2\sigma^2}},\; dv = x e^{-\frac{x^2}{2\sigma^2}} dx\right\} = \\ = \dfrac{1}{\sigma^2}\left[ x\cdot \int x e^{-\frac{x^2}{2\sigma^2}} dx \big|_0^\infty - \int_0^\infty x e^{-\frac{x^2}{2\sigma^2}} dx \right] = \left\{ \int x e^{-bx^2} dx = \dfrac{1}{2b} e^{-bx^2} \right\} = \dfrac{1}{\sigma^2} \left[ x \cdot \dfrac{1}{2} (-2\sigma^2) e^{-\frac{x^2}{2\sigma^2}} \bigg|_0^\infty - \int_0^\infty \dfrac{1}{2}(-2\sigma^2)e^{-\frac{x^2}{2\sigma^2}}dx\right] = \\ = 0 + \int\limits_0^\infty e^{-\frac{x^2}{2\sigma^2}}dx = \dfrac{\sqrt{2 \pi \sigma^2}}{\sqrt{2 \pi \sigma^2}}\cdot \int\limits_0^\infty e^{-\frac{x^2}{2\sigma^2}} dx = \dfrac{\sqrt{2 \pi \sigma^2}}{2} = \sigma\sqrt{\dfrac{\pi}{2}}$

  • $E(\xi)^2 = \int\limits_0^\infty x^2 \dfrac{x}{\sigma ^ 2} e^{-\frac{x^2}{2\sigma^2}} dx = \left\{u = x^2, \; du = 2dx, \; v = \int x e^{-\frac{x^2}{2\sigma^2}}, \; dv = x e^{-\frac{x^2}{2\sigma^2}} dx\right\} = x^2\dfrac{1}{2\sigma^2}\left(-2\sigma^2\right) e^{-\frac{x^2}{2\sigma^2}} \bigg|_0^\infty - \\ - \int\limits_0^\infty 2x \dfrac{1}{2\sigma^2}\dfrac{1}{2}\left(-2\sigma^2\right) e^{-\frac{x^2}{2\sigma^2}} dx = 0 + 2 \int\limits_0^\infty x e^{-\frac{x^2}{2\sigma^2}} dx = 2 \left(-2\sigma^2\right) \dfrac{1}{2} e^{-\frac{x^2}{2\sigma^2}} \bigg|_0^\infty = 0 - \left(-2\sigma^2\right) = 2\sigma^2$
